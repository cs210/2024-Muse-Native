# 2024-Muse-Native

To get it working: 

```
npm i
npm i expo
npx expo install @supabase/supabase-js @react-native-async-storage/async-storage react-native-url-polyfill
```
First time (or after major changes):
```
npx expo run:ios
```
Every other time (run command then press 'i'):
```
npx expo start
```
